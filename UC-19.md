# Use Case UC-19: View Registration Prices

## Goal
Public users view published registration prices with final CAD amounts and no discount computation.

## Preconditions
- Pricing has been published.

## Main Flow
1. User opens registration prices page.
2. System loads published pricing snapshot.
3. System returns CAD formatted final amounts.
4. System indicates missing-information markers for incomplete categories.

## Extensions
- Unpublished pricing: 404 not available.
- Retrieval failure: 500 safe error.
- Temporary failure: subsequent successful retry returns prices.

## Success End Condition
Public user sees stable published CAD prices.

## Failure End Condition
No unpublished pricing or discount artifacts are exposed.
