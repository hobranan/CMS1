# Use Case UC-19: View Registration Prices

**Goal in Context**: Public users view published registration prices with final CAD amounts and no discount computation.  
**Scope**: Conference Management System (CMS)  
**Level**: User Goal  
**Primary Actor**: User  
**Secondary Actors**: CMS services, data store  
**Trigger**: The primary actor initiates this use case.

## Success End Condition
* Public user sees stable published CAD prices.

## Failed End Condition
* The use case does not complete, and the system returns an explicit, actionable failure state.

## Preconditions
* Pricing has been published.

## Main Success Scenario
1. Primary actor performs the primary action for this use case.
2. System returns expected success outcome.

## Extensions
* **1a**: Unpublished pricing: 404 not available.
* **1b**: Retrieval failure: 500 safe error.
* **1c**: Temporary failure: subsequent successful retry returns prices.
