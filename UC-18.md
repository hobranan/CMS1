# Use Case UC-18: View Final Conference Schedule (Public PDF)

**Goal in Context**: Any public user can view the final published schedule and open/download a PDF version.  
**Scope**: Conference Management System (CMS)  
**Level**: User Goal  
**Primary Actor**: User  
**Secondary Actors**: CMS services, data store  
**Trigger**: The primary actor initiates this use case.

## Success End Condition
* Public user can browse published schedule and retrieve PDF.

## Failed End Condition
* The use case does not complete, and the system returns an explicit, actionable failure state.

## Preconditions
* Conference schedule has been published.

## Main Success Scenario
1. Primary actor performs the primary action for this use case.
2. System returns expected success outcome.

## Extensions
- Schedule not published: return not-available (404).
- Detail entry missing: return not-found.
- Retrieval/PDF failures: return safe 500 error.
