# Use Case UC-18: View Final Conference Schedule (Public PDF)

## Goal
Any public user can view the final published schedule and open/download a PDF version.

## Preconditions
- Conference schedule has been published.

## Main Flow
1. User opens public schedule URL.
2. System verifies schedule is published.
3. System returns grouped day/session schedule data.
4. User opens session detail.
5. System returns detail with policy-based field visibility and unavailable markers.
6. User requests PDF (inline or attachment).
7. System returns PDF artifact.

## Extensions
- Schedule not published: return not-available (404).
- Detail entry missing: return not-found.
- Retrieval/PDF failures: return safe 500 error.

## Success End Condition
Public user can browse published schedule and retrieve PDF.

## Failure End Condition
No unpublished or restricted data is leaked; user receives explicit error response.
